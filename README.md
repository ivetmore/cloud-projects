# Cloud Projects Portfolio

This repository serves as my cloud engineering learning portfolio, and contains a collection of projects focused on learning cloud infrastructure, containerization, and DevOps practices. Each project demostrates different concepts related to deploying and managing applications in cloud environments.

## Projects

### Docker Web App 

A simple containerized web application built with Docker.

**Concepts practiced:**
• Docker containerization
• Dockerfile creation
• Building and running containers locally
 
### Docker Flask Redis (AWS Deployment)

A containerized Flask API deployed to AWS using ECS (Fargate), an Application Load Balancer, and ElastiCache (Redis).

**Concepts practiced:**
• Docker containerization
• AWS ECS (Fargate) deployment
• Amazon ECR image storage
• Application Load Balancer configuration
• ElastiCache (Redis) integration
• Cloud networking and security groups

Live demo: http://docker-flask-redis-alb-1836018808.us-east-2-elb.amazonaws.com
